Se corre con *flutter run* en web, para verlo en mobil :


Widgets : Son la unidad basica de la interfaz de usuario

Arbol de widgets : Es la estructura de la aplicacion, donde cada widget es un nodo del arbol

El import package:flutter/material.dart es para importar los widgets de material design, que son los que se utilizan para crear la interfaz de usuario

StatelessWidget : Es un widget que no tiene estado, es decir, que no cambia con el tiempo

StatefulWidget : Es un widget que tiene estado, es decir, que cambia con el tiempo

void main() : Es el punto de entrada de la aplicacion

void runApp() : Es la funcion que se encarga de ejecutar la aplicacion

const SmarturApp() : Es el widget raiz de la aplicacion

const MaterialApp() : Es el cascaron de la app

const Scaffold() : Es la estructura de la pantalla, scaffold es el esqueleto de la pantalla

const AppBar() : Es la barra de la pantalla

const Center() : Es el centro de la pantalla

const Text() : Es el texto de la pantalla
